***Read this file before running the software.***
Articulate is a software to prepare articles on single or multiple topics at once. It saves the articles in MS-Word automatically with a little formatting. Welcome to articulate.

***How to run***
1. You will need Python 3 installed on your system to run this software.
2. You will need to install the following modules to run Articulate
	a. PyQt5
	b. python-docx
	c. Wikipedia
3. Once the above modules are installed run article_maker.py. It will prompt to enter topics. Enter single or multiple topics with comma "," and spaces on both sides on comma.
4. It will prepare the docx in the directory where article_maker.py is present.

In order to prepare package of this software you will need to use setup.py.
